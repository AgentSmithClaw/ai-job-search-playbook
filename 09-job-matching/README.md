# 09-job-matching — 岗位匹配与机会筛选

## 📌 与 GapPilot Platform 的关系

本模块是 **GapPilot 方法论的手动执行版**，提供岗位匹配规则、评分逻辑、投递跟踪等框架工具。

核心功能已产品化为 **[GapPilot Platform](https://github.com/AgentSmithClaw/ai-job-search-platform)**，支持：
- 上传简历 + 粘贴 JD → AI 自动分析匹配度
- 多维度机会评分（岗位匹配、公司吸引力、成长空间等）
- 结构化 gap 报告输出

| | 本模块（方法论版） | GapPilot Platform（产品版） |
|---|---|---|
| **形态** | Markdown 模板 + 人工判断 | Web 应用 + AI 自动分析 |
| **使用场景** | 理解底层逻辑、手动评分 | 日常投递决策、批量筛选 |
| **适合** | 学习框架、自定义规则 | 高效执行、标准化流程 |

**建议**：先读本模块理解方法论，日常使用 GapPilot Platform 提升效率。
