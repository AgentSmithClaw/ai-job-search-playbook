# Inbox

> 所有暂时还没分类、但值得先留下的信息，先放这里。

## 使用规则
- 先记录，后整理，不要卡在“该放哪里”。
- 每条信息尽量简短，写清来源、日期、下一步。
- 每周至少清理一次，把高价值内容转移到正式模块。

## 建议格式
```markdown
- [YYYY-MM-DD] 标题
  - 来源：
  - 内容：
  - 下一步：
```

## 去向参考
- 岗位/公司相关 → `01-job-target/`
- 学习计划/学习笔记 → `02-learning-plan/`
- 面试问题/STAR案例 → `03-interview-prep/`
- 项目经验/项目复盘 → `04-projects/`
- 简历/作品集材料 → `05-portfolio/` 或 `08-resume-optimization/`
- 过程记录/阶段结论 → `06-logs/`
