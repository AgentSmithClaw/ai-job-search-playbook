# GitHub 首页介绍

> 打造专业的 GitHub 个人主页

---

## 基础信息

### 个人头像
- 使用真实照片或专业头像
- 保持一致的品牌形象

### 用户名
- 使用真实姓名或常用昵称
- 保持简洁易记

### Bio
- 一句话介绍自己
- 突出核心技能或方向

示例：
```
AI应用研究者 | 需求分析专家 | 项目推进
```

---

## README 内容

### 1. 头部 Banner
```markdown
<h1>你好，我是 [名字]</h1>

<p>专注于 [领域] 的 [职位]</p>
```

### 2. 关于我
- 教育背景
- 工作经历
- 技术专长
- 当前兴趣

### 3. 技术栈
- 熟练: Python, FastAPI, SQL
- 了解: React, Docker

### 4. 项目展示
- 突出 2-3 个核心项目
- 包含项目描述和技术栈

### 5. 统计
- 使用 GitHub Stats Card
- 展示贡献和活跃度
