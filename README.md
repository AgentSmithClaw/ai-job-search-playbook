# GapPilot Playbook

> 个人求职方法论与知识体系 —— 从岗位分析到面试复盘的完整操作系统。

## 项目背景

这套 Playbook 是我在求职过程中沉淀的方法论体系：岗位拆解、差距分析、证据映射、面试准备、学习计划、投递管理。它的核心价值不是"整理文档"，而是**把求职从感性判断变成可执行的分析流程**。

后来我把其中"JD 与简历匹配"的核心能力产品化成了 [GapPilot Platform](https://github.com/AgentSmithClaw/ai-job-search-platform)，一个 AI 驱动的求职差距分析平台。

**两个项目的关系：Playbook 是方法论，Platform 是产品化。**

- 📘 GapPilot Playbook — 你正在看的这个：方法论、面试准备、项目复盘
- ⚡ [GapPilot Platform](https://github.com/AgentSmithClaw/ai-job-search-platform) — AI 驱动的求职分析产品

## 当前目标

- 目标岗位：AI研究助理（广州 4399）
- 管理内容：岗位分析、学习路线、面试准备、作品项目、进度日志、关键决策

## 目录结构

- `01-job-target/` 岗位目标、JD分析、差距分析、公司研究
- `02-learning-plan/` 3个月计划、6个月计划、周计划、学习笔记
- `03-interview-prep/` 自我介绍、问答库、STAR案例、模拟面试
- `04-projects/` 作品项目总览与项目链接
- `05-portfolio/` 作品集文案、GitHub展示规划
- `06-logs/` 进度日志、问答摘要、周报
- `07-assets/` 截图、图表、草稿素材
- `08-resume-optimization/` 主简历、定制简历、项目描述优化
- `09-job-matching/` 岗位匹配、投递跟踪、沟通话术（核心算法已产品化为 [Platform](https://github.com/AgentSmithClaw/ai-job-search-platform)）
