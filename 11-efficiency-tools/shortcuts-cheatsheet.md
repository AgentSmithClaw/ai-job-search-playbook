# 快捷键 Cheat Sheet

> 常用快捷键速查

---

## Mac 常用

### 通用
| 操作 | 快捷键 |
|------|--------|
| 复制 | ⌘C |
| 粘贴 | ⌘V |
| 撤销 | ⌘Z |
| 全选 | ⌘A |
| 搜索 | ⌘F |
| 关闭窗口 | ⌘W |
| 退出 | ⌘Q |

### 截图
| 操作 | 快捷键 |
|------|--------|
| 全屏截图 | ⌘⇧3 |
| 区域截图 | ⌘⇧4 |
| 窗口截图 | ⌘⇧4 → 空格 |

## VSCode

### 基础编辑
| 操作 | 快捷键 |
|------|--------|
| 命令面板 | ⌘⇧P |
| 快速打开 | ⌘P |
| 切换标签 | ⌘W |
| 分屏 | ⌘\ |
| 终端 | ⌘` |

### 代码编辑
| 操作 | 快捷键 |
|------|--------|
| 移动行 | ⌥↑/↓ |
| 复制行 | ⌥⇧↑/↓ |
| 多行编辑 | ⌘⇧L |
| 注释 | ⌘/ |
