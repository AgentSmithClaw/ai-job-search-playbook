# VSCode 插件推荐清单

> 提升开发效率的必备插件

---

## 通用插件

### 美化与主题
- **One Dark Pro** - 经典暗色主题
- **Dracula Theme** - 优雅暗色主题
- **Material Icon Theme** - 文件图标美化

### 编辑增强
- **Prettier** - 代码格式化
- **ESLint** - 代码检查
- **Auto Rename Tag** - 自动重命名标签
- **Auto Close Tag** - 自动闭合标签
- **Bracket Pair Colorizer** - 括号着色

---

## Python 开发

### 必备
- **Python** - Python 语言支持
- **Pylance** - Python 语言服务器
- **Jupyter** - Jupyter Notebook 支持

### 辅助
- **Python Docstring Generator** - 文档生成
- **Black Formatter** - Python 格式化

---

## Git 集成

- **GitLens** - Git 增强
- **Git Graph** - Git 提交图
- **Gitmoji** - Gitmoji 支持

---

## AI/ML

- **Copilot** - AI 代码补全
- **TensorFlow Snippets** - TensorFlow 代码片段

---

## 效率工具

- **Thunder Client** - API 测试
- **REST Client** - REST API 测试
- **JSON Tools** - JSON 格式化

## 快捷键技巧

| 操作 | Mac | Windows |
|------|-----|---------|
| 打开命令面板 | Cmd+Shift+P | Ctrl+Shift+P |
| 快速打开文件 | Cmd+P | Ctrl+P |
| 多行选择 | Cmd+Shift+L | Ctrl+Shift+L |
| 格式化代码 | Shift+Alt+F | Shift+Alt+F |
