# 入职前检查清单

> 拿到Offer后到入职前需要完成的事项

---

## 1. 确认Offer

### 收到Offer后
- [ ] 仔细阅读Offer内容
- [ ] 确认薪资、职级、入职时间
- [ ] 确认福利待遇
- [ ] 确认工作地点

### 签署前
- [ ] 如有疑问，及时沟通
- [ ] 争取合理权益
- [ ] 书面确认（邮件）

---

## 2. 离职手续

### 原公司
- [ ] 提交辞职信
- [ ] 办理工作交接
- [ ] 归还公司物品
- [ ] 结算工资
- [ ] 开具离职证明

### 时间安排
- [ ] 确认最后工作日
- [ ] 预留交接时间（通常2周）

---

## 3. 入职准备

### 证件准备
- [ ] 身份证原件及复印件
- [ ] 学历证书原件及复印件
- [ ] 上一家公司离职证明
- [ ] 体检报告（如需要）
- [ ] 银行卡（公司指定银行）

### 物品准备
- [ ] 电脑（如需自备）
- [ ] 办公用品

---

## 4. 档案与社保

### 档案
- [ ] 确认档案存放地点
- [ ] 办理档案转移（如需要）

### 社保
- [ ] 确认社保缴纳时间
- [ ] 办理社保转移（如需要）

---

## 5. 心理准备

### 心态调整
- [ ] 保持空杯心态
- [ ] 设定合理预期
- [ ] 准备迎接新挑战

### 目标设定
- [ ] 设定30天目标
- [ ] 了解部门工作流程
- [ ] 认识新同事

---

## 6. 其他

- [ ] 告知亲友新工作安排
- [ ] 更新简历状态（如已在线）
- [ ] 准备第一天工作计划
