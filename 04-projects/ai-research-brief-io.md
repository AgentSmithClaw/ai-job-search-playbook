# ai-research-brief-generator 输入输出样例

> 项目2数据流转示例

---

## 输入样例

### 资讯输入（示例）
```
来源：巴比特
标题：以太坊Gas费用创历史新低
内容：近期以太坊网络Gas费用持续下降，平均Gas价格降至10 Gwei以下，
创2021年以来新低...
```

### 多来源资讯包（JSON格式）
```json
{
  "source": "巴比特",
  "timestamp": "2026-03-12",
  "category": "市场",
  "title": "以太坊Gas费用创历史新低",
  "content": "...",
  "tags": ["以太坊", "Gas", "市场"]
}
```

---

## 处理链路

### Step1: 清洗与去重
- 去除HTML标签
- 去除重复内容
- 标准化日期格式

### Step2: 主题归类
- 市场动态
- 技术进展
- 监管政策
- 竞品动态

### Step3: 摘要生成
- 提取核心信息（100字内）
- 提取关键数据点
- 标注来源可信度

### Step4: 建议输出
- 行动建议
- 风险提示
- 后续关注点

## 字段映射表
| 输入字段 | 处理后字段 | 说明 |
