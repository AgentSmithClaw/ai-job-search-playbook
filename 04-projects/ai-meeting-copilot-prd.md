# ai-meeting-copilot PRD（补全版）

> AI会议纪要与行动项助手 - 产品需求文档

---

## 1. 产品概述

### 1.1 产品定位
面向团队协同场景的AI会议纪要与行动项管理工具，解决会后信息分散、行动项不清晰、历史难追溯的问题。

### 1.2 目标用户
- 项目团队负责人
- 跨部门协作对接人
- 需要定期会议跟进的知识工作者

### 1.3 核心价值
- 将会议内容结构化：结论、行动项、风险、下一步
- 行动项可追踪：状态、责任人、截止时间
- 历史可查询：按会议/时间/状态检索

---

## 2. 功能架构

### 2.1 核心功能
| 功能 | 描述 | 优先级 |
|---|---|---|
| 会议输入 | 支持文本/语音/文件导入 | P0 |
| AI结构化 | 自动提取结论/行动项/风险 | P0 |
| 行动项管理 | 状态流转（待办→进行→完成→阻塞） | P0 |
| 历史会议 | 按时间/关键词检索 | P1 |
| 仪表盘 | 统计：总数、完成率、超期率 | P1 |
| 提醒机制 | 截止前提醒/超期提醒 | P2 |

### 2.2 数据结构
```
会议记录
├── id
├── 标题
├── 原始内容
├── 会议时间
├── 结论（数组）
├── 行动项（数组）
│   ├── 内容
│   ├── 责任人
│   ├── 截止时间
│   └── 状态
├── 风险（数组）
└── 下一步

行动项状态
- 待办（todo）
- 进行中（in_progress）
- 阻塞（blocked）
- 完成（done）
```

---

## 3. MVP边界

### 3.1 MVP必须完成（P0）
- 会议输入（文本）
- AI结构化生成（结论+行动项）
- 行动项状态管理（4状态）
- 历史记录查看

### 3.2 MVP不包含（P1+）
- 语音输入
- 自动提醒
- 多语言支持
- 团队协作（多人权限）

---

## 4. 核心指标

### 4.1 业务指标
| 指标 | 目标 |
|---|---|
| 会议纪要结构化率 | ≥90% |
| 行动项闭环率 | ≥80% |
| 用户满意度 | ≥4/5 |

### 4.2 技术指标
| 指标 | 目标 |
|---|---|
| 响应时间 | <3秒 |
| 结构化准确率 | ≥85% |
